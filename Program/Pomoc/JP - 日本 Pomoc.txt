Ctrl + D - デモ
Ctrl + N - 新しいゲーム
Ctrl + P - ヒント
Ctrl + T - シャッフル
Ctrl + X - ボードをクリアします
Ctrl + W - 終了
Ctrl + A - キューブの表示
Ctrl + Z - 元に戻す
Ctrl + O - オプション

Alt + Enter - 全画面表示/全画面表示を終了
Esc - 全画面表示を終了

0 - カメラを中央に配置します
9 - カメラの初期位置
K、スペース、RMB - マウスを動かしながらカメラを動かす
Z - ズームパネルの表示/非表示

Del - カメラをリセットします
Ctrl + Del - カメラが指しているポイントをリセットします

Ctrl + 1 - 時間表示なし
Ctrl + 2 - ゲーム時間を表示します
Ctrl + 3 - 現在の日付と時刻を表示します

Ctrl + + - キューブを増やす
Ctrl + - - キューブを縮小する
Ctrl + Backspace - キューブのデフォルトサイズ

[ - 立方体の厚さを減らす
] - 立方体の厚さを増やします

;、Shift +;  - カメラの動きの速度を下げる
'、Shift +' - カメラの移動速度を上げる

Ctrl + I - 削除するペアの数を表示します

Ctrl + B - ボードの背景画像を表示します


Ctrl + , - クロックサイクルを延長します
Ctrl + .  - クロックサイクルを短くします
Ctrl + / - デフォルトのクロックサイクル


テンキー
/ - ポイントに向けられたカメラ/ボードに垂直

Del - カメラをリセットします
Ctrl + Del - カメラが指しているポイントをリセットします

数字 - 視点の変化
Ctrl + 数値 - カメラが見ているポイントを移動します
